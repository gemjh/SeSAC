# UML Diagram Generation Requirements
# This file contains system dependencies and tools needed for UML sequence diagram generation

# ============================================================================
# System Dependencies (macOS - via Homebrew)
# ============================================================================
# The following packages need to be installed via Homebrew:
# 
# brew install plantuml
# 
# PlantUML dependencies that are automatically installed:
# - openjdk (Java runtime required for PlantUML)
# - graphviz (For enhanced diagram rendering)
# - Various image processing libraries:
#   - libavif
#   - gd  
#   - jasper
#   - netpbm
#   - gts
#   - gdk-pixbuf
#   - librsvg
#   - m4
#   - libtool

# ============================================================================
# Installation Commands
# ============================================================================

# macOS (Homebrew):
# brew install plantuml

# Ubuntu/Debian:
# sudo apt-get update
# sudo apt-get install plantuml

# CentOS/RHEL/Fedora:
# sudo yum install plantuml
# # or for newer versions:
# sudo dnf install plantuml

# Windows:
# - Install Java JRE/JDK
# - Download plantuml.jar from https://plantuml.com/download
# - Use: java -jar plantuml.jar -tpng your_diagram.puml

# ============================================================================
# Usage
# ============================================================================
# Convert PlantUML file to PNG:
# plantuml -tpng clap_system_uml_sequence.puml

# Convert to SVG:
# plantuml -tsvg clap_system_uml_sequence.puml

# Convert to PDF:
# plantuml -tpdf clap_system_uml_sequence.puml
